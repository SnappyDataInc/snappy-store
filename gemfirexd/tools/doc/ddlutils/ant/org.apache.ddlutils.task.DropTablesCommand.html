dropTables

Description

Java type: org.apache.ddlutils.task.DropTablesCommand

Sub task for dropping tables.

Parameters

tables : String

Sets the names of the tables to be removed, as a comma-separated list. Escape a comma via '\,' if it is part of the table name. Please note that table names are not trimmed which means that whitespace characters should only be present in this string if they are actually part of the table name (i.e. in delimited identifer mode).
If no table filter is specified, then all tables will be dropped.

failonerror : boolean

Specifies whether the execution shall stop if an error has occurred during the task runs.
By default execution will be stopped when an error is encountered.

tablefilter : String

Sets the regular expression matching the names of the tables to be removed. For case insensitive matching, an uppercase name can be assumed. If no regular expressionis specified
If no table filter is specified, then all tables will be dropped.
